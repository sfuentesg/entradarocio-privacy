Política de Privacidad de EntradaRocio

Última actualización: 7 de mayo de 2026

Esta política de privacidad describe cómo Serafin Fuentes Gutierrez (“nosotros”, “nuestro”) recopila, usa y protege la información cuando utiliza la aplicación móvil EntradaRocio (en adelante, “la Aplicación”).

1. Información que recopilamos

La Aplicación utiliza los siguientes datos para su funcionamiento:

  • Cámara del dispositivo: se accede a la cámara exclusivamente para escanear matrículas de vehículos en tiempo real. Las imágenes capturadas no se almacenan ni se envían a ningún servidor; el procesamiento se realiza localmente en el dispositivo.
  • Matrícula del vehículo: el texto detectado (o introducido manualmente) se envía a la API de control de acceso (api.micharret.com) para consultar si el vehículo tiene autorización de entrada. Esta comunicación se realiza mediante HTTPS.

2. Uso de la información

La información procesada se utiliza exclusivamente para:

  • Verificar si un vehículo dispone de pase de acceso vigente.
  • Registrar los accesos autorizados en el sistema de gestión.

No utilizamos los datos para publicidad, análisis de comportamiento ni los compartimos con terceros salvo el sistema de gestión propio descrito anteriormente.

3. Permisos solicitados

  • android.permission.CAMERA: necesario para escanear matrículas mediante la cámara del dispositivo. Sin este permiso, el escáner automático no estará disponible, aunque podrá introducir la matrícula de forma manual.
  • android.permission.INTERNET: necesario para consultar la API de control de acceso.

4. Almacenamiento y seguridad

La Aplicación no almacena imágenes ni vídeo en el dispositivo ni en servidores externos. Toda comunicación con la API se realiza sobre conexiones cifradas (HTTPS/TLS).

5. Menores de edad

Esta Aplicación está destinada a uso profesional por parte de personal autorizado. No está dirigida a menores de 18 años ni recopilamos conscientemente datos de menores.

6. Cambios en esta política

Podemos actualizar esta política ocasionalmente. Te notificaremos de cambios significativos publicando la nueva versión en esta misma página.
